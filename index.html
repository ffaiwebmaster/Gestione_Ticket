What makes it different

Built for real teams,
not for DevOps engineers.

Most self-hosted tools assume you have a server, a terminal, and half a weekend. Gestionale Lavorazioni assumes you have a job to do.

▣

Single-file architecture

The entire application — UI, logic, and database layer — lives in one PHP file. Upload it, open it, done.

◈

SQLite database

No MySQL, no PostgreSQL. The database is a single file next to your PHP. Backup is just copy-paste.

◎

5-step setup wizard

First launch opens a guided wizard. Set your organization name, sector, logo, and you're running in under two minutes.

⬡

Works anywhere PHP runs

Shared hosting, VPS, local Windows with a .bat launcher, Cloudflare Tunnel. If PHP 8 runs there, this runs there.

◐

Dynamic branding

Upload your logo, set your app name. The interface reflects your organization, not the software vendor.

⊕

Italian / English UI

Full i18n support built in. Switch between Italian and English from the settings panel.

Interface preview

See it before you run it.

Three views from the actual interface. Same dark tone, same clarity, on every device.

Get started

Running in 60 seconds.

Choose your environment. The steps are the same everywhere: copy the file, open the URL, follow the wizard.

Windows
Shared Hosting
VPS / Linux
# 1. Download the ZIP and extract it
# 2. Double-click AVVIA.bat — PHP built-in server starts
# 3. Browser opens automatically at:
http://localhost:8080/gestionale_lavorazioni.php
# 4. Follow the 5-step setup wizard
# 1. Upload gestionale_lavorazioni.php via FTP to public_html/
# 2. Make sure the folder is writable (chmod 755)
# 3. Open the URL in your browser:
https://yourdomain.com/gestionale_lavorazioni.php
# 4. Follow the 5-step setup wizard
# Apache: point DocumentRoot to the folder containing the file
# nginx: see the sample config in the repo README
cp gestionale_lavorazioni.php /var/www/html/
chown www-data:www-data /var/www/html/gestionale_lavorazioni.php
# Open https://yourdomain.com/gestionale_lavorazioni.php

First launch

The setup wizard.

No manual configuration. The wizard collects everything it needs on first run and never asks again.

01

Organization name

Your company or team name. Appears in the header and all documents.

02

Sector selection

Choose from 7 preconfigured profiles or define a custom sector. Terminology and categories adapt automatically.

03

Logo upload

Upload your logo. It's stored as base64 in the SQLite database — no extra files to manage.

04

Admin credentials

Set your username and password. Access control is built in from day one.

05

Ready

The wizard completes and redirects to your fully configured instance. No restart required.
